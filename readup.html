Flashcard Maker Documentation



Flashcard Maker and Flashcard Reader are twin programs.



Flashcard Reader requires Flashcard Maker, but Flashcard Maker doesn't require Flashcard Reader, so Flashcard Maker can be used for other websites.



Flashcard Reader uses its own format to store flashcards and read it, so Import From Quizlet will convert it to the Flashcard Reader format.



Flashcard Maker can be used on GimKit, Quizlet, and other flashcard websites.



The source code is available on github.
